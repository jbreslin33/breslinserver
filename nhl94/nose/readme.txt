INSTALLATION
************

 - Extract the NOSE distribution into your preferred folder and execute nose.exe

 - If prompted, you may be required to download and install the Visual Basic
   6.0 Service Pack 5 Run-time distribution files (vbrun60sp5.exe).  It can be
   found at:

   http://search.microsoft.com/search/results.aspx?qu=vbrun60sp5.exe

 - Additionally, you may be required to download and install the Windows Common
   Controls ActiveX Control DLL (mscomctl.ocx). The latest version can be
   found at:

   http://activex.microsoft.com/controls/vb6/mscomctl.cab

   Simply extract the contents of the downloaded CAB file using your favourite
   unzip program supporting CAB files and double-click the mscomctl.inf to
   install the control.
